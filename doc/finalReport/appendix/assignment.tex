\section{细节性工作}
\begin{description}
  \item[自动测试脚本] 搭建基于 \url{travis.com} 的持续集成（自动测试流程）。可以利用已有的分片试验结果，在每一次 \texttt{Push} 和 \texttt{Pull Request} 时对代码进行全面的自动测试。这部分工作由管唯宇同学完成。
  \item[DOF~自动适配] 在兼容已有输入格式的基础上，可以更灵活地适应不同的旋转自由度约束要求。这部分工作由邓博元、杨正宇、黄云帆、管唯宇四位同学共同完善而成。
  \item[若干 bug 修正] 修正原计时模块在多线程下会出现的计时不正确 bug，修正 debug 模式下打印刚度阵会出现的访问越界 bug，修正多单元组时出现的基类数组转换 bug 等。这部分工作由管唯宇同学完成。
  \item[8H~刚度阵组装] 由于此前调用~Eigen~库进行运算的效率过低，故改写为显式写法，大幅提高了效率。这部分工作由黄云帆同学在陈一彤同学原来代码的基础上稍作修改后完成。
  \item[STAP90~程序] 为已有用~Fortran90~语言编写的~STAP90~程序的完善，增加了若干单元和后处理功能。这部分工作由卢晟昊同学完成。
\end{description}
